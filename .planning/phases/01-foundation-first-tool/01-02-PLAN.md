     - `void start(int port = 6800)` -- creates TCPServer, calls listen(). Default port 6800 per user decision.
     - `void stop()` -- closes server and client connection
     - `void poll()` -- called from _process(), checks for new connections, reads available data, dispatches messages
     - `bool is_running() const`
   - Private members:
     - `godot::Ref<godot::TCPServer> tcp_server`
     - `godot::Ref<godot::StreamPeerTCP> client_peer`
     - `std::string read_buffer` -- accumulates TCP data for newline-delimited framing
     - `bool initialized` -- tracks whether MCP handshake completed
     - `int port`
   - Private methods:
     - `void process_message(const std::string& line)` -- parse JSON-RPC, route to handler
     - `void send_response(const nlohmann::json& response)` -- serialize to JSON string + newline, write to client_peer
     - `std::string handle_request(const std::string& method, const nlohmann::json& id, const nlohmann::json& params)` -- dispatch by method name

2. Create `src/mcp_server.cpp`:
   - `start()`: Instantiate TCPServer via `memnew`, call `listen(port)`. Log to Godot console via `UtilityFunctions::print()`.
   - `stop()`: Disconnect client_peer if valid, stop tcp_server.
   - `poll()`:
     - If tcp_server has connection available, accept it via `take_connection()`. Only allow one client at a time (disconnect previous if new one arrives). Reset `initialized` state.
     - If client_peer is valid and connected:
       - Call `client_peer->poll()` (required for StreamPeerTCP to update state)
       - Check `get_available_bytes()`. If > 0, read with `get_data()`, append to `read_buffer`.
       - Split `read_buffer` on newline (`\n`). For each complete line, call `process_message(line)`. Keep any incomplete trailing data in `read_buffer`.
   - `process_message(line)`:
     - Call `parse_jsonrpc(line)` from mcp_protocol.h
     - If parse failed, send error response
     - If notification (e.g., "notifications/initialized"), handle silently (set initialized=true for this notification)
     - If request, call `handle_request(method, id, params)`
   - `handle_request(method, id, params)`:
     - "initialize" -> return `create_initialize_response(id)`
     - "tools/list" -> return `create_tools_list_response(id)`
     - "tools/call" -> extract tool name from params["name"]:
       - "get_scene_tree" -> extract optional arguments from params["arguments"]: max_depth (int, default -1 = unlimited), include_properties (bool, default true), root_path (string, default ""). Call `get_scene_tree(max_depth, include_properties, root_path)` from scene_tools.h, return `create_tool_result(id, result)`
       - unknown tool -> return `create_tool_not_found_error(id, tool_name)`
     - unknown method -> return `create_error_response(id, METHOD_NOT_FOUND, "Method not found: " + method)`
   - `send_response(json)`: serialize with `.dump()` (no pretty print), append "\n", convert to PackedByteArray, write via `client_peer->put_data()`
   - IMPORTANT: TCP message framing uses newline-delimited JSON, matching MCP stdio transport. Accumulate in read_buffer and split on '\n'.

   - `serialize_node(Node* node, int current_depth, int max_depth, bool include_properties)`:
     - Extract name: `node->get_name().utf8().get_data()`
     - Extract type: `node->get_class().utf8().get_data()`
     - Extract path: `node->get_path().operator String().utf8().get_data()`
     - If include_properties is true, also extract per user decision:
       - **transform**: For Node2D: extract position (x,y), rotation (radians), scale (x,y) via `get_position()`, `get_rotation()`, `get_scale()`. For Node3D: extract position (x,y,z), rotation (x,y,z in radians), scale (x,y,z) via `get_position()`, `get_rotation()`, `get_scale()`. For other nodes: omit transform field.
       - **visible**: For CanvasItem (2D): `is_visible()`. For Node3D: `is_visible()`. For other nodes: omit visible field.
       - **has_script**: `!node->get_script().is_null()` (check if script is attached)
       - **script_path**: If has_script, get script resource path via `node->get_script().call("get_path")` or cast to Script and call `get_path()`. Use `Ref<Script> script = node->get_script(); if (script.is_valid()) script->get_path().utf8().get_data()`. If no script, omit script_path field.
     - Recursively serialize children: if max_depth == -1 (unlimited) or current_depth < max_depth, loop `get_child_count()`, call `serialize_node(get_child(i), current_depth + 1, max_depth, include_properties)`.      - Otherwise, return `serialize_node(start_node, 0, max_depth, include_properties)`
   - Follow the exact pattern from research Code Examples (Scene Tree Recursive Traversal) but extended with the additional properties.
