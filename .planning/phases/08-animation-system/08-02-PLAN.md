**Helper 1: AnimPlayerLookupResult + lookup_animation_player()**
- Struct with `bool success`, `nlohmann::json error`, `AnimationPlayer* player`
- Gets scene root from `EditorInterface::get_singleton()->get_edited_scene_root()`
- Returns error JSON if no scene open, node not found, or node is not AnimationPlayer
- Uses `Object::cast_to<AnimationPlayer>(node)`

**Helper 2: find_animation()**
- Takes `AnimationPlayer* player, const std::string& anim_name`
- Uses `player->has_animation(StringName(anim_name.c_str()))` to check existence
- Returns `player->get_animation(StringName(anim_name.c_str()))` (Ref<Animation>)
- Returns invalid Ref if not found

**Helper 3: track_type_map**
- Static `std::unordered_map<std::string, Animation::TrackType>` with 4 entries:
  - "value" -> Animation::TYPE_VALUE
  - "position_3d" -> Animation::TYPE_POSITION_3D
  - "rotation_3d" -> Animation::TYPE_ROTATION_3D
  - "scale_3d" -> Animation::TYPE_SCALE_3D

**Helper 4: track_type_to_string()**
- Reverse mapping from Animation::TrackType to string for query output

**Helper 5: loop_mode_from_string() / loop_mode_to_string()**
- "none" -> Animation::LOOP_NONE, "linear" -> Animation::LOOP_LINEAR, "pingpong" -> Animation::LOOP_PINGPONG
- Return -1 for unknown string

**Helper 6: variant_to_string()**
- Uses `UtilityFunctions::var_to_str(val)` to convert any Variant to string
- Returns `std::string(godot_str.utf8().get_data())`

**Function 1: create_animation()**
Per CONTEXT.md: Creates AnimationPlayer + AnimationLibrary + Animation. UndoRedo supported.
- If `player_path` is non-empty: lookup existing player, skip node creation
- If `player_path` is empty: create new AnimationPlayer via `ClassDB::instantiate(StringName("AnimationPlayer"))`
  - Set name to `node_name` (default "AnimationPlayer")
  - Find parent node (parent_path or scene root)
  - UndoRedo: `create_action("MCP: Create AnimationPlayer")`, `add_do_method(parent, "add_child", player, true)`, `add_do_method(player, "set_owner", scene_root)`, `add_do_reference(player)`, `add_undo_method(parent, "remove_child", player)`, `commit_action()`
- Ensure default library exists: check `has_animation_library(StringName(""))`, if not, instantiate and add
- Create Animation: `Ref<Animation> anim; anim.instantiate(); anim->set_length(1.0);`
- Add to library: `lib->add_animation(StringName(animation_name.c_str()), anim)`
- If animation already exists: return error JSON
- Return: `{"success": true, "player_path": <computed_path>, "animation_name": animation_name}`
- Compute player_path using: `scene_root->get_path_to(player)` converted to string via `String(...).utf8().get_data()`

**Function 2: add_animation_track()**
Per CONTEXT.md: No UndoRedo.
- Lookup player via lookup_animation_player()
- Find animation via find_animation()
- Validate track_type against track_type_map, return error if unknown
- `int track_idx = anim->add_track(it->second);`
- `anim->track_set_path(track_idx, NodePath(track_path.c_str()));`
- Return: `{"success": true, "track_index": track_idx, "track_type": track_type, "track_path": track_path}`

**Function 3: set_keyframe()**
Per CONTEXT.md: No UndoRedo. Uses action parameter: "insert", "update", "remove".
- Lookup player and animation
- Validate track_index bounds: `if (track_index < 0 || track_index >= anim->get_track_count())` return error
- Get track type: `Animation::TrackType type = anim->track_get_type(track_index);`

For "insert":
- Require value parameter (non-empty), parse via `parse_variant(value, nullptr, "")`
- Switch on track type:
  - TYPE_VALUE: `anim->track_insert_key(track_index, time, parsed_value, transition)`
  - TYPE_POSITION_3D: `anim->position_track_insert_key(track_index, time, Vector3(parsed_value))`
  - TYPE_ROTATION_3D: `anim->rotation_track_insert_key(track_index, time, Quaternion(parsed_value))`
  - TYPE_SCALE_3D: `anim->scale_track_insert_key(track_index, time, Vector3(parsed_value))`
- Return: `{"success": true, "action": "insert", "track_index": track_index, "time": time}`

For "update":
- Require value parameter, parse via parse_variant
- Find key at time: `int key_idx = anim->track_find_key(track_index, time, Animation::FIND_MODE_APPROX);`
- If key_idx == -1: return error "No keyframe found at time X on track Y"
- `anim->track_set_key_value(track_index, key_idx, parsed_value)`
- If transition param provided (not default): `anim->track_set_key_transition(track_index, key_idx, transition)`
- Return: `{"success": true, "action": "update", "track_index": track_index, "time": time}`

For "remove":
- Find key at time: `int key_idx = anim->track_find_key(track_index, time, Animation::FIND_MODE_APPROX);`
- If key_idx == -1: return error
- `anim->track_remove_key(track_index, key_idx)`
- Return: `{"success": true, "action": "remove", "track_index": track_index, "time": time}`

For unknown action: return error "Unknown action: X. Valid: insert, update, remove"

**Function 4: get_animation_info()**
Per CONTEXT.md: Full depth query.
- Lookup player
- If animation_name is non-empty: query only that animation
- If animation_name is empty: iterate all via `player->get_animation_list()`
- For each animation:
  - Get Ref<Animation> via `player->get_animation(StringName(name.c_str()))`
  - Build JSON: `{"name": name, "length": anim->get_length(), "loop_mode": loop_mode_to_string(anim->get_loop_mode()), "step": anim->get_step(), "track_count": anim->get_track_count()}`
  - For each track (0..track_count-1):
    - `{"index": i, "type": track_type_to_string(anim->track_get_type(i)), "path": String(anim->track_get_path(i)).utf8().get_data(), "key_count": anim->track_get_key_count(i)}`
    - For each key (0..key_count-1):
      - `{"time": anim->track_get_key_time(i, k), "value": variant_to_string(anim->track_get_key_value(i, k)), "transition": anim->track_get_key_transition(i, k)}`
- Return: `{"success": true, "player_path": player_path, "animations": [...]}`

**Function 5: set_animation_properties()**
Per CONTEXT.md: UndoRedo supported.
- Lookup player, find animation
- `undo_redo->create_action(String("MCP: Set Animation Properties"))`
- If props contains "length" (number):
  - `double old_length = anim->get_length();`
  - `undo_redo->add_do_method(anim.ptr(), "set_length", props["length"].get<double>())`
  - `undo_redo->add_undo_method(anim.ptr(), "set_length", old_length)`
- If props contains "loop_mode" (string):
  - Parse via loop_mode_from_string(), return error if unknown
  - `int old_mode = anim->get_loop_mode();`
  - `undo_redo->add_do_method(anim.ptr(), "set_loop_mode", new_mode)`
  - `undo_redo->add_undo_method(anim.ptr(), "set_loop_mode", old_mode)`
- If props contains "step" (number):
  - `double old_step = anim->get_step();`
  - `undo_redo->add_do_method(anim.ptr(), "set_step", props["step"].get<double>())`
  - `undo_redo->add_undo_method(anim.ptr(), "set_step", old_step)`
- `undo_redo->commit_action()`
- Return: `{"success": true, "animation_name": animation_name}`
