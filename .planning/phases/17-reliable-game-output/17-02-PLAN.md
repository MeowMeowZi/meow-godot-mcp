**Test structure:**

2. **Test 2: Run game with bridge** -- `run_game(mode="main", wait_for_bridge=true)`, verify success + bridge_connected
3. **Test 3: Print output capture via eval** -- Use `eval_in_game(expression="print('meow_test_17')")` to trigger a print, then sleep 1 second (well within the 200ms poll + processing margin), then `get_game_output()`. Assert the result contains a line with "meow_test_17".

7. **Test 7: Keyword filtering** -- `get_game_output(clear_after_read=false, keyword="meow_test_17")`. Should return only lines containing that keyword.
8. **Test 8: clear_after_read incremental** -- Call `get_game_output(clear_after_read=true)` to drain. Then `eval_in_game(expression="print('incremental_17')")`, sleep 0.5s, `get_game_output()`. Should contain "incremental_17" but NOT any previous test prints.
