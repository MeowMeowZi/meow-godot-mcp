Make get_game_output reliably capture print() output from running games by having the companion GDScript actively forward log content through the debugger channel, eliminating dependency on file_logging flush timing.

Purpose: The current approach uses _has_capture("output") which Godot 4.6 does not deliver to EditorDebuggerPlugin, and the file-based fallback has unreliable flush timing. This plan adds companion-side log file reading and forwarding via the existing meow_mcp debugger channel, which IS reliable.

Output: Companion script with log forwarding, editor-side game_log handler, cleaned up dispatch logic.
</objective>

**Companion GDScript changes (meow_mcp_bridge.gd):**

Add a log file reader that runs in _process() and forwards new lines via the debugger channel:

1. Add member variables at class level:
   - `var _log_file_path: String = ""` (resolved in _ready)
   - `var _log_read_pos: int = 0` (file byte offset for incremental reads)
   - `var _log_poll_timer: float = 0.0` (accumulator for polling interval)
   - `const LOG_POLL_INTERVAL: float = 0.2` (200ms polling - ensures sub-1s capture)

2. In `_ready()`, after the existing EngineDebugger setup:
   - Resolve log path: `_log_file_path = OS.get_user_data_dir().path_join("logs/godot.log")`
   - Initialize read position to current file end: open the file, seek to end, save position, close
   - Enable `_process(true)` (set_process)

3. Add `func _process(delta: float)`:
   - Accumulate `_log_poll_timer += delta`
   - If `_log_poll_timer < LOG_POLL_INTERVAL`, return early
   - Reset timer: `_log_poll_timer = 0.0`
   - Call `_forward_new_log_lines()`

4. Add `func _forward_new_log_lines()`:
   - If `_log_file_path.is_empty()` or not `FileAccess.file_exists(_log_file_path)`, return
   - Open file: `var f = FileAccess.open(_log_file_path, FileAccess.READ)`
   - If f is null, return
   - Get file length: `var file_len = f.get_length()`
   - If `file_len <= _log_read_pos`, close and return (no new data)
   - Seek to `_log_read_pos`: `f.seek(_log_read_pos)`
   - Read remaining text: `var new_text = f.get_buffer(file_len - _log_read_pos).get_string_from_utf8()`
   - Update position: `_log_read_pos = file_len`
   - Close file
   - Split into lines: `var lines = new_text.split("\n", false)` (skip empty)
   - If lines is empty, return
   - For each line, classify level:
     - If line begins with "ERROR:" or "   at:" or "   GDScript backtrace" -> "error"
     - If line begins with "WARNING:" -> "warning"
     - Else -> "info"
   - Build arrays: `var texts: Array = []` and `var levels: Array = []`
   - For each classified line, append to texts and levels
   - Send via debugger: `EngineDebugger.send_message("meow_mcp:game_log", [texts, levels])`

```cpp
    // Companion forwards log lines from game's log file
    if (p_data.size() >= 2) {
        auto now_ms = std::chrono::duration_cast<std::chrono::milliseconds>(
            std::chrono::steady_clock::now().time_since_epoch()).count();

        Array texts = p_data[0];
        Array levels = p_data[1];
        for (int i = 0; i < texts.size(); i++) {
            String text = texts[i];
            std::string msg(text.utf8().get_data());
            // Remove trailing \r if present
            if (!msg.empty() && msg.back() == '\r') msg.pop_back();

                        level = lvl_str;
                    }
                }
                log_buffer.push_back({msg, level, now_ms});

        }
    }
    return true;
}
```

Replace the current get_game_output dispatch (around line 593-643) with:

```cpp
if (tool_name == "get_game_output") {
    bool clear_after_read = true;
    std::string level;
    int64_t since = 0;
    std::string keyword;
    if (params.contains("arguments") && params["arguments"].is_object()) {
        auto& args = params["arguments"];
        if (args.contains("clear_after_read") && args["clear_after_read"].is_boolean())
            clear_after_read = args["clear_after_read"].get<bool>();
        if (args.contains("level") && args["level"].is_string())
            level = args["level"].get<std::string>();
        if (args.contains("since") && args["since"].is_number_integer())
            since = args["since"].get<int64_t>();
        if (args.contains("keyword") && args["keyword"].is_string())
            keyword = args["keyword"].get<std::string>();
    }
    // Use debugger-channel buffer when bridge is active (companion forwards log data)
    if (game_bridge && game_bridge->is_game_connected()) {
        auto result = game_bridge->get_buffered_game_output(clear_after_read, level, since, keyword);
        return mcp::create_tool_result(id, result);
    }
    // File-based fallback only when bridge is not available
    auto file_result = get_game_output(clear_after_read);
    nlohmann::json structured = nlohmann::json::array();
    if (file_result.contains("lines")) {
        for (auto& line : file_result["lines"]) {
            std::string text = line.is_string() ? line.get<std::string>() : "";
            std::string inferred_level = "info";
            if (text.find("ERROR:") == 0 || text.find("   at:") == 0 ||
                text.find("   GDScript backtrace") == 0 ||
                text.find("       [") == 0) {
                inferred_level = "error";
            } else if (text.find("WARNING:") == 0) {
                inferred_level = "warning";
            }
            if (!level.empty() && inferred_level != level) continue;
            if (!keyword.empty() && text.find(keyword) == std::string::npos) continue;
            structured.push_back({{"text", text}, {"level", inferred_level}, {"timestamp_ms", 0}});
        }
    }
    file_result["lines"] = structured;
    file_result["count"] = structured.size();
    file_result["total_buffered"] = file_result.value("count", 0);
    return mcp::create_tool_result(id, file_result);
}
```
