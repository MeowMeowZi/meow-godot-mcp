    Create with `#ifndef MEOW_GODOT_MCP_COMPOSITE_TOOLS_H` guard. Include nlohmann/json and string.

    Pure C++ function (testable without Godot):
    - `bool find_nodes_match_name(const std::string& pattern, const std::string& name)` -- glob-style matching supporting `*` wildcards. Case-insensitive. If pattern has no `*`, do substring match.

    Godot-dependent functions (guarded by `#ifdef MEOW_GODOT_MCP_GODOT_ENABLED`):
    - `nlohmann::json find_nodes(const std::string& type, const std::string& name_pattern, const std::string& property_name, const std::string& property_value, const std::string& root_path)` -- Traverse scene tree recursively from root (or root_path). For each node: check type match (ClassDB::is_parent_class for inheritance), name pattern match (find_nodes_match_name), property value match (node->get() and string comparison). Return `{"nodes": [...], "count": N}` where each entry has `"path"` and `"type"`. All filter params are optional; omitted/empty means "don't filter on this".
    - `nlohmann::json batch_set_property(const nlohmann::json& node_paths, const std::string& type_filter, const std::string& property_name, const std::string& value_str, godot::EditorUndoRedoManager* undo_redo)` -- Two modes: if `node_paths` is a JSON array of strings, iterate those paths. If `type_filter` is non-empty, use find_nodes logic to collect matching nodes. For each matched node: store old property value, set new value via `parse_variant` + `node->set()`. Wrap entire batch in single UndoRedo action: `create_action("MCP: Batch Set Property")`, for each node add `add_do_method(node, "set", property, new_value)` and `add_undo_method(node, "set", property, old_value)`, then `commit_action()`. Return `{"modified": N, "errors": [...]}`.

    **Implementation (src/composite_tools.cpp):**
    - `find_nodes_match_name`: Convert both pattern and name to lowercase. If pattern contains no `*`, do `name.find(pattern) != npos`. If pattern starts and ends with `*`, substring search on middle. If pattern ends with `*`, prefix match. If pattern starts with `*`, suffix match. For general glob patterns with `*`, split on `*` and check parts appear in order.
    - `find_nodes`: Get `EditorInterface::get_singleton()->get_edited_scene_root()`. If null, return `{"error": "No scene open"}`. Recursive helper `find_nodes_recursive(Node* node, Node* scene_root, ...)` that builds the result array. For type matching: if `type` is non-empty, use `ClassDB::is_parent_class(node->get_class(), StringName(type.c_str()))` OR exact match `node->get_class() == StringName(type.c_str())` to support inheritance (e.g., type="Control" matches Label, Button, etc.). For property matching: if `property_name` is non-empty, get `node->get(StringName(property_name.c_str()))` and compare `String(variant).utf8().get_data()` to `property_value`.
    - `batch_set_property`: Validate inputs. If both node_paths and type_filter are empty, return error. Collect target nodes (either from explicit paths or find by type). Use UndoRedo pattern from physics_tools.cpp.

    ```cpp
    if (tool_name == "find_nodes") {
        auto& args = get_args(params);
        std::string type = get_string(args, "type");
        std::string name_pattern = get_string(args, "name_pattern");
        std::string property_name, property_value;
        if (args.contains("property_filter") && args["property_filter"].is_object()) {
            property_name = get_string(args["property_filter"], "name");
            property_value = get_string(args["property_filter"], "value");
        }
        std::string root_path = get_string(args, "root_path");
        if (type.empty() && name_pattern.empty() && property_name.empty()) {
            return make_params_error(id, "At least one search filter required: type, name_pattern, or property_filter", tool_name);
        }
        return make_tool_response(id, find_nodes(type, name_pattern, property_name, property_value, root_path), tool_name);
    }

    if (tool_name == "batch_set_property") {
        auto& args = get_args(params);
        nlohmann::json node_paths = args.contains("node_paths") && args["node_paths"].is_array() ? args["node_paths"] : nlohmann::json();
        std::string type_filter = get_string(args, "type_filter");
        std::string property = get_string(args, "property");
        std::string value = get_string(args, "value");
        if (property.empty() || value.empty()) {
            return make_params_error(id, "Missing required parameters: property, value", tool_name);
        }
        if (node_paths.empty() && type_filter.empty()) {
            return make_params_error(id, "Must provide node_paths array or type_filter", tool_name);
        }
        return make_tool_response(id, batch_set_property(node_paths, type_filter, property, value, undo_redo), tool_name);
    }
    ```
