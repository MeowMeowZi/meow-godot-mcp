    **Step 1: Create `src/scene_mutation.h`:**
    ```cpp
    #ifndef GODOT_MCP_MEOW_SCENE_MUTATION_H
    #define GODOT_MCP_MEOW_SCENE_MUTATION_H

    #include <nlohmann/json.hpp>
    #include <string>

    namespace godot {
    class EditorUndoRedoManager;
    }

    // Create a new node in the scene tree with undo/redo support
    // type: Node class name (e.g., "Sprite2D", "CharacterBody3D")
    // parent_path: path to parent node (empty = scene root)
    // name: desired node name (Godot may auto-rename for uniqueness)
    // properties: optional key-value pairs to set after creation
    // Returns JSON: {"success": true, "path": "Parent/NewNode"} or {"error": "..."}
    nlohmann::json create_node(const std::string& type, const std::string& parent_path,
                                const std::string& name, const nlohmann::json& properties,
                                godot::EditorUndoRedoManager* undo_redo);

    // Set a property on an existing node with undo/redo support
    // node_path: path to target node (relative to scene root)
    // property: property name in snake_case (e.g., "position", "visible", "modulate")
    // value_str: property value as string (auto-parsed by variant_parser)
    // Returns JSON: {"success": true} or {"error": "..."}
    nlohmann::json set_node_property(const std::string& node_path, const std::string& property,
                                      const std::string& value_str,
                                      godot::EditorUndoRedoManager* undo_redo);

    // Delete a node from the scene tree with undo/redo support
    // node_path: path to the node to delete (relative to scene root)
    // Cannot delete the scene root.
    // Returns JSON: {"success": true} or {"error": "..."}
    nlohmann::json delete_node(const std::string& node_path,
                                godot::EditorUndoRedoManager* undo_redo);

    #endif // GODOT_MCP_MEOW_SCENE_MUTATION_H
    ```

    **`create_node` implementation:**
    1. Get scene root: `Node* scene_root = EditorInterface::get_singleton()->get_edited_scene_root();`
    2. If `!scene_root` return `{"error", "No scene open"}`
    3. If `!undo_redo` return `{"error", "UndoRedo not available"}`
    4. Validate class: `ClassDB::get_singleton()->class_exists(StringName(type.c_str()))` -- if false return `{"error", "Unknown class: " + type}`
    5. Validate is Node: `ClassDB::get_singleton()->is_parent_class(StringName(type.c_str()), StringName("Node"))` -- if false return `{"error", type + " is not a Node type"}`
    6. Find parent: if `parent_path` empty use `scene_root`, else `scene_root->get_node_or_null(NodePath(parent_path.c_str()))` -- if null return `{"error", "Parent not found: " + parent_path}`
    7. Instantiate: `Variant instance = ClassDB::get_singleton()->instantiate(StringName(type.c_str()));` then `Node* new_node = Object::cast_to<Node>(instance.operator Object*());` -- if null return `{"error", "Failed to instantiate: " + type}`
    8. Set name: if `name` not empty, `new_node->set_name(String(name.c_str()))`, else use type as name
    9. UndoRedo action:
       ```cpp
       undo_redo->create_action(String("MCP: Create ") + String(type.c_str()));
       undo_redo->add_do_method(parent, "add_child", new_node, true);
       undo_redo->add_do_method(new_node, "set_owner", scene_root);
       undo_redo->add_do_reference(new_node);
       undo_redo->add_undo_method(parent, "remove_child", new_node);
       ```
    10. If `properties` is a JSON object and not empty, iterate over key-value pairs and for each:
        ```cpp
        Variant parsed_val = parse_variant(val_str, new_node, key);
        Variant old_val = new_node->get(StringName(key.c_str()));
        undo_redo->add_do_property(new_node, StringName(key.c_str()), parsed_val);
        undo_redo->add_undo_property(new_node, StringName(key.c_str()), old_val);
        ```
        NOTE: Properties are set AFTER add_child/set_owner within the same undo action. This is important because some properties may only work correctly on nodes in the tree.
    11. `undo_redo->commit_action();`
    12. Return: `{"success": true, "path": String(scene_root->get_path_to(new_node)).utf8().get_data(), "type": type}`

    **`set_node_property` implementation:**
    1. Get scene root, check not null
    2. Find node: `scene_root->get_node_or_null(NodePath(node_path.c_str()))` -- if null return error
    3. If `!undo_redo` return error
    4. Parse value: `Variant new_value = parse_variant(value_str, node, property);`
    5. Get old value: `Variant old_value = node->get(StringName(property.c_str()));`
    6. UndoRedo:
       ```cpp
       undo_redo->create_action(String("MCP: Set ") + String(property.c_str()));
       undo_redo->add_do_property(node, StringName(property.c_str()), new_value);
       undo_redo->add_undo_property(node, StringName(property.c_str()), old_value);
       undo_redo->commit_action();
       ```
    7. Return `{"success": true}`

    **`delete_node` implementation:**
    1. Get scene root, check not null
    2. Find node -- if null return error
    3. If `node == scene_root` return `{"error", "Cannot delete scene root"}`
    4. If `!undo_redo` return error
    5. Get parent: `Node* parent = node->get_parent();`
    6. Get node index for undo ordering: `int index = node->get_index();` (for restoring position)
    7. UndoRedo:
       ```cpp
       undo_redo->create_action(String("MCP: Delete ") + String(node->get_name()));
       undo_redo->add_do_method(parent, "remove_child", node);
       undo_redo->add_undo_method(parent, "add_child", node, true);
       undo_redo->add_undo_method(node, "set_owner", scene_root);
       undo_redo->add_undo_reference(node);
       undo_redo->commit_action();
       ```
    8. Return `{"success": true}`

        }
