<task type="checkpoint:human-verify" gate="blocking">
  <name>Task 2: End-to-end UAT for Scene CRUD tools</name>
  <what-built>
    Three new MCP tools for scene manipulation:
    1. **create_node** -- creates nodes by class name with optional parent, name, and initial properties
    2. **set_node_property** -- modifies any property on an existing node with type auto-parsing
    3. **delete_node** -- removes a node by path
    All operations are integrated with Godot's undo/redo system.
  </what-built>
  <how-to-verify>
    **Prerequisites:**
    - Godot 4.3+ editor with the project open
    - Plugin enabled in Project Settings > Plugins > Godot MCP Meow
    - Create a simple test scene (Node2D root called "TestRoot") and save it
    - Bridge executable configured or use direct TCP for testing

    **Test 1: tools/list shows new tools (SCNE-02/03/04)**
    Send: `{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-03-26","capabilities":{},"clientInfo":{"name":"test","version":"1.0"}}}`
    Then: `{"jsonrpc":"2.0","method":"notifications/initialized"}`
    Then: `{"jsonrpc":"2.0","id":2,"method":"tools/list"}`
    Expected: Response contains 4 tools: get_scene_tree, create_node, set_node_property, delete_node

    **Test 2: create_node basic (SCNE-02)**
    Send: `{"jsonrpc":"2.0","id":3,"method":"tools/call","params":{"name":"create_node","arguments":{"type":"Sprite2D","name":"TestSprite"}}}`
    Expected: success=true, path contains "TestSprite"
    Verify: Node appears in Scene panel under TestRoot

    **Test 3: create_node with parent path (SCNE-02)**
    Send: `{"jsonrpc":"2.0","id":4,"method":"tools/call","params":{"name":"create_node","arguments":{"type":"Node2D","parent_path":"TestSprite","name":"Child"}}}`
    Expected: success=true, path contains "TestSprite/Child"
    Verify: Child node appears under TestSprite in Scene panel

    **Test 4: create_node with initial properties (SCNE-02 + SCNE-06)**
    Send: `{"jsonrpc":"2.0","id":5,"method":"tools/call","params":{"name":"create_node","arguments":{"type":"Sprite2D","name":"ColorSprite","properties":{"modulate":"#ff0000","visible":"false"}}}}`
    Expected: success=true
    Verify: Node has red modulate and is invisible in Inspector

    **Test 5: set_node_property position (SCNE-03 + SCNE-06)**
    Send: `{"jsonrpc":"2.0","id":6,"method":"tools/call","params":{"name":"set_node_property","arguments":{"node_path":"TestSprite","property":"position","value":"Vector2(100, 200)"}}}`
    Expected: success=true
    Verify: TestSprite position shows (100, 200) in Inspector

    **Test 6: set_node_property visibility (SCNE-03)**
    Send: `{"jsonrpc":"2.0","id":7,"method":"tools/call","params":{"name":"set_node_property","arguments":{"node_path":"TestSprite","property":"visible","value":"false"}}}`
    Expected: success=true
    Verify: TestSprite shows as hidden in Scene panel (eye icon dimmed)

    **Test 7: delete_node (SCNE-04)**
    Send: `{"jsonrpc":"2.0","id":8,"method":"tools/call","params":{"name":"delete_node","arguments":{"node_path":"TestSprite/Child"}}}`
    Expected: success=true
    Verify: Child node removed from Scene panel

    **Test 8: delete_node scene root protection (SCNE-04)**
    Send: `{"jsonrpc":"2.0","id":9,"method":"tools/call","params":{"name":"delete_node","arguments":{"node_path":"TestRoot"}}}`
    Expected: error containing "Cannot delete scene root"

    **Test 9: Undo/Redo (SCNE-05)**
    Press Ctrl+Z three times (should undo: delete Child, set visible=false, set position)
    Verify: Child reappears under TestSprite, TestSprite visible again, position back to original
    Press Ctrl+Y three times
    Verify: Changes reapply in order

    **Test 10: Invalid class error (SCNE-02)**
    Send: `{"jsonrpc":"2.0","id":10,"method":"tools/call","params":{"name":"create_node","arguments":{"type":"FakeNodeType"}}}`
    Expected: error containing "Unknown class"

    **Test 11: Node not found error (SCNE-03)**
    Send: `{"jsonrpc":"2.0","id":11,"method":"tools/call","params":{"name":"set_node_property","arguments":{"node_path":"NonExistent","property":"visible","value":"true"}}}`
    Expected: error containing "not found"
  </how-to-verify>
