<task type="auto">
  <name>Task 1: Build interactive test scene via MCP tools</name>
  <files>project/subagent_test.tscn, project/subagent_test.gd</files>
  <action>
Use Godot MCP tools in this sequence to build the scene entirely through MCP calls. Do NOT write .tscn files directly.

1. **Create scene** with a Control root:
   - `mcp__godot__create_scene` with node_type="Control", node_name="SubagentTest"

2. **Set root to full_rect**:
   - `mcp__godot__set_layout_preset` with node_path="SubagentTest", preset="full_rect"

3. **Add a VBoxContainer** as child of root for vertical layout:
   - `mcp__godot__create_node` with parent_path="SubagentTest", node_type="VBoxContainer", node_name="Layout"

4. **Center the VBoxContainer**:
   - `mcp__godot__set_layout_preset` with node_path="SubagentTest/Layout", preset="center"

5. **Configure VBoxContainer spacing**:
   - `mcp__godot__set_container_layout` with node_path="SubagentTest/Layout", separation=20, alignment="center"

6. **Add a Label**:
   - `mcp__godot__create_node` with parent_path="SubagentTest/Layout", node_type="Label", node_name="StatusLabel"

7. **Set Label text**:
   - `mcp__godot__set_node_property` with node_path="SubagentTest/Layout/StatusLabel", property="text", value="MCP Subagent Test - Press the button!"

8. **Set Label font size and center alignment**:
   - `mcp__godot__set_theme_override` with node_path="SubagentTest/Layout/StatusLabel", overrides={"font_size": 32}
   - `mcp__godot__set_node_property` with node_path="SubagentTest/Layout/StatusLabel", property="horizontal_alignment", value=1

9. **Add a Button**:
   - `mcp__godot__create_node` with parent_path="SubagentTest/Layout", node_type="Button", node_name="TestButton"

10. **Set Button text and minimum size**:
    - `mcp__godot__set_node_property` with node_path="SubagentTest/Layout/TestButton", property="text", value="Click Me!"
    - `mcp__godot__set_node_property` with node_path="SubagentTest/Layout/TestButton", property="custom_minimum_size", value="200,50"

11. **Write the GDScript** that handles button press:
    - `mcp__godot__write_script` with path="res://subagent_test.gd", content:
      ```gdscript
      extends Control

      var click_count: int = 0

      func _ready() -> void:
          print("Subagent MCP Test: Scene loaded successfully!")

      func _on_test_button_pressed() -> void:
          click_count += 1
          $Layout/StatusLabel.text = "MCP works! Clicked %d time(s)" % click_count
          print("Button pressed! Count: %d" % click_count)
      ```

12. **Attach script** to root node:
    - `mcp__godot__attach_script` with node_path="SubagentTest", script_path="res://subagent_test.gd"

13. **Connect button signal**:
    - `mcp__godot__connect_signal` with source_path="SubagentTest/Layout/TestButton", signal_name="pressed", target_path="SubagentTest", method_name="_on_test_button_pressed"

14. **Save the scene**:
    - `mcp__godot__save_scene` with path="res://subagent_test.tscn"
  </action>
