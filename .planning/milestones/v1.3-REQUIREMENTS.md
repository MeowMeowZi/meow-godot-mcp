# Requirements: v1.3 Developer Experience Polish

## DX — Developer Experience

| ID | Requirement | Priority |
|----|-------------|----------|
| DX-01 | `run_game` 支持 `wait_for_bridge` 参数，自动等待 bridge 连接后返回（超时报错） | Must |
| DX-02 | 所有接受 `node_path` 的工具统一支持 `""` 和 `"."` 表示场景根节点 | Must |
| DX-03 | `get_game_output` 在所有场景下可靠捕获 print() 输出（≤1秒延迟） | Must |
| DX-04 | `set_layout_preset` 支持对场景根节点使用 | Should |

**Motivation:** v1.2 实际使用中发现的摩擦点——bridge 连接需要手动 sleep 轮询、根节点路径不统一、game output 在某些场景返回空
