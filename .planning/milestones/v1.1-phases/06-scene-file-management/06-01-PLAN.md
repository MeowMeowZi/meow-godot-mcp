The 5 new tool definitions:

1. **save_scene** (SCNF-01 + SCNF-05):
   - Description: `"Save the current scene to disk. Without path: overwrites current file (Ctrl+S). With path: saves to new location (Ctrl+Shift+S). Returns error if scene has no file path and no path is provided."`
   - Properties:
     - `path` (string, optional): `"Optional file path (e.g., res://scenes/level.tscn). Omit to overwrite current file. Extension determines format: .tscn (text) or .scn (binary)."`
   - Required: `nlohmann::json::array()` (no required params -- path is optional)

2. **open_scene** (SCNF-02):
   - Description: `"Open an existing scene file in the editor. Adds a new tab without closing the current scene. The opened scene becomes the active edited scene."`
   - Properties:
     - `path` (string): `"Path to scene file (e.g., res://scenes/level.tscn)"`
   - Required: `{"path"}`

3. **list_open_scenes** (SCNF-03):
   - Description: `"List all currently open scenes in the editor. Returns file paths, titles, and which scene is active."`
   - Properties: `nlohmann::json::object()` (no properties)
   - Required: `nlohmann::json::array()` (no required params)

4. **create_scene** (SCNF-04):
   - Description: `"Create a new scene with the specified root node type, save it to disk, and open it in the editor. The root node class must be a Node subclass."`
   - Properties:
     - `root_type` (string): `"Root node class name (e.g., Node2D, Node3D, Control, CharacterBody2D)"`
     - `path` (string): `"File path to save the scene (e.g., res://scenes/player.tscn)"`
     - `root_name` (string, optional): `"Name for the root node. Defaults to the class name if omitted."`
   - Required: `{"root_type", "path"}`

5. **instantiate_scene** (SCNF-06):
   - Description: `"Instantiate a PackedScene (.tscn/.scn) as a child node in the current scene. Supports undo/redo. The instance root gets its owner set to the scene root."`
   - Properties:
     - `scene_path` (string): `"Path to the scene file to instantiate (e.g., res://scenes/enemy.tscn)"`
     - `parent_path` (string, optional): `"Path to parent node relative to scene root. Empty or omit for scene root."`
     - `name` (string, optional): `"Override name for the instantiated node. Defaults to the scene file name if omitted."`
   - Required: `{"scene_path"}`
