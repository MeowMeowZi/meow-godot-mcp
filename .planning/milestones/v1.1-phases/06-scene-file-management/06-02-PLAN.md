Implement 5 functions:

**1. save_scene(path)** -- SCNF-01 + SCNF-05:
- Get EditorInterface singleton.
- Get edited scene root via `ei->get_edited_scene_root()`. If null, return `{"error", "No scene open"}`.
- If `path` is empty (overwrite mode):
  - Get `root->get_scene_file_path()`. If empty, return `{"error", "Scene has no file path. Provide a path parameter to save as new file."}`.
  - Call `Error err = ei->save_scene()`. If `err != OK`, return error with error code.
  - Return `{"success": true, "path": <scene_file_path_as_std_string>}`.
- If `path` is non-empty (save-as mode):
  - Call `validate_res_path(path, error_out)`. If invalid, return `{"error": error_out}`.
  - Call `ei->save_scene_as(String(path.c_str()), true)`.
  - Verify file exists after save via `FileAccess::file_exists(String(path.c_str()))`. If not, return `{"error", "Failed to save scene to: " + path}`.
  - Return `{"success": true, "path": path}`.

**2. open_scene(path)** -- SCNF-02:
- Call `validate_res_path(path, error_out)`. If invalid, return error.
- Check file exists: `FileAccess::file_exists(String(path.c_str()))`. If not, return `{"error", "Scene file not found: " + path}`.
- Call `EditorInterface::get_singleton()->open_scene_from_path(String(path.c_str()))`.
- Return `{"success": true, "path": path}`.

**3. list_open_scenes()** -- SCNF-03:
- Get `EditorInterface* ei = EditorInterface::get_singleton()`.
- Get `PackedStringArray scenes = ei->get_open_scenes()`.
- Get `Node* active_root = ei->get_edited_scene_root()`.
- Get `String active_path = active_root ? active_root->get_scene_file_path() : String()`.
- Build JSON array: for each scene path, extract title (filename from path using rfind('/')), determine is_active by comparing with active_path.
- Return `{"success": true, "scenes": <array>, "count": <int>}`.

**4. create_scene(root_type, path, root_name)** -- SCNF-04:
- Validate class exists: `ClassDB::class_exists(StringName(root_type.c_str()))`. If not, return `{"error", "Unknown class: " + root_type}`.
- Validate is Node subclass: `ClassDB::is_parent_class(StringName(root_type.c_str()), StringName("Node"))`. If not, return `{"error", root_type + " is not a Node type"}`.
- Validate path: `validate_res_path(path, error_out)`. If invalid, return error.
- Instantiate root: `Variant instance = ClassDB::instantiate(StringName(root_type.c_str()))`. Cast to `Node*`. If null, return error.
- Set name: `root->set_name(String(root_name.empty() ? root_type.c_str() : root_name.c_str()))`.
- Pack: `Ref<PackedScene> packed; packed.instantiate(); Error pack_err = packed->pack(root);`.
- CRITICAL: Free temporary node with `memdelete(root)` (NOT queue_free -- node is not in scene tree).
- If pack_err != OK, return error.
- Save: `Error save_err = ResourceSaver::get_singleton()->save(packed, String(path.c_str()))`. If save_err != OK, return error.
- Open in editor: `EditorInterface::get_singleton()->open_scene_from_path(String(path.c_str()))`.
- Return `{"success": true, "path": path, "root_type": root_type}`.

**5. instantiate_scene(scene_path, parent_path, name, undo_redo)** -- SCNF-06:
- If `!undo_redo`, return `{"error", "UndoRedo not available"}`.
- Get scene root: `Node* scene_root = EditorInterface::get_singleton()->get_edited_scene_root()`. If null, return `{"error", "No scene open"}`.
- Validate scene_path: `validate_res_path(scene_path, error_out)`. If invalid, return error.
- Check file exists: `FileAccess::file_exists(String(scene_path.c_str()))`. If not, return `{"error", "Scene file not found: " + scene_path}`.
- Load PackedScene: `Ref<Resource> res = ResourceLoader::get_singleton()->load(String(scene_path.c_str()), "PackedScene")`. Cast: `Ref<PackedScene> packed = res`. If `!packed.is_valid()`, return `{"error", "Failed to load scene: " + scene_path}`.
- Instantiate with editor state: `Node* instance = packed->instantiate(PackedScene::GEN_EDIT_STATE_INSTANCE)`. If null, return error.
- If `name` is non-empty, set name: `instance->set_name(String(name.c_str()))`.
- Find parent node: if `parent_path` empty, parent = scene_root. Otherwise `Node* parent = scene_root->get_node_or_null(NodePath(parent_path.c_str()))`. If null, return `{"error", "Parent not found: " + parent_path}` (also `memdelete(instance)` before returning).
- UndoRedo (same pattern as create_node in scene_mutation.cpp):
  ```cpp
  undo_redo->create_action(String("MCP: Instantiate scene"));
  undo_redo->add_do_method(parent, "add_child", instance, true);
  undo_redo->add_do_method(instance, "set_owner", scene_root);
  undo_redo->add_do_reference(instance);
  undo_redo->add_undo_method(parent, "remove_child", instance);
  undo_redo->commit_action();
  ```
- Build node path using manual path construction (same as create_node pattern -- avoids get_path_to null errors):
  ```cpp
  std::string node_name = std::string(String(instance->get_name()).utf8().get_data());
  std::string actual_path;
  if (parent == scene_root) {
      actual_path = node_name;
  } else {
      actual_path = parent_path + "/" + node_name;
  }
  ```
- Return `{"success": true, "path": actual_path, "scene_path": scene_path}`.

All string conversions use the established pattern: `std::string(some_godot_String.utf8().get_data())` for Godot String to std::string.
