**Tool 1: create_animation**
- name: "create_animation"
- description: "Create an AnimationPlayer node with an AnimationLibrary and a named Animation resource. If player_path is provided, adds the animation to an existing AnimationPlayer instead of creating a new one."
- input_schema properties:
  - `animation_name` (string, required): "Name for the new Animation resource"
  - `player_path` (string, optional): "Path to existing AnimationPlayer node. If omitted, creates a new AnimationPlayer."
  - `parent_path` (string, optional): "Parent node path for new AnimationPlayer (default: scene root). Ignored if player_path is set."
  - `node_name` (string, optional): "Name for the new AnimationPlayer node (default: AnimationPlayer). Ignored if player_path is set."
- required: ["animation_name"]

**Tool 2: add_animation_track**
- name: "add_animation_track"
- description: "Add a typed track to an existing Animation. Returns the track index for use with set_keyframe."
- input_schema properties:
  - `player_path` (string, required): "Path to AnimationPlayer node relative to scene root"
  - `animation_name` (string, required): "Name of the Animation to add the track to"
  - `track_type` (string, required): "Track type: value, position_3d, rotation_3d, or scale_3d"
  - `track_path` (string, required): "Node path the track targets, relative to AnimationPlayer (e.g. '../Sprite2D:position')"
- required: ["player_path", "animation_name", "track_type", "track_path"]

**Tool 3: set_keyframe**
- name: "set_keyframe"
- description: "Insert, update, or remove a keyframe on an animation track. Values are parsed as strings (e.g. 'Vector3(1,0,0)', '0.5', 'Color(1,0,0,1)')."
- input_schema properties:
  - `player_path` (string, required): "Path to AnimationPlayer node relative to scene root"
  - `animation_name` (string, required): "Name of the Animation containing the track"
  - `track_index` (integer, required): "Track index (returned by add_animation_track)"
  - `time` (number, required): "Time in seconds for the keyframe"
  - `action` (string, required): "Action to perform: insert, update, or remove"
  - `value` (string, optional): "Keyframe value as string (required for insert/update, ignored for remove)"
  - `transition` (number, optional): "Transition easing value (default: 1.0 = linear). Values <1 ease-in, >1 ease-out."
- required: ["player_path", "animation_name", "track_index", "time", "action"]

**Tool 4: get_animation_info**
- name: "get_animation_info"
- description: "Query an AnimationPlayer's animations, track structure, and keyframe data. Returns full depth: animation list with per-animation tracks and per-track keyframes."
- input_schema properties:
  - `player_path` (string, required): "Path to AnimationPlayer node relative to scene root"
  - `animation_name` (string, optional): "Query a specific animation only. If omitted, returns all animations."
- required: ["player_path"]

**Tool 5: set_animation_properties**
- name: "set_animation_properties"
- description: "Set Animation resource properties: duration, loop mode, and step. Supports undo/redo."
- input_schema properties:
  - `player_path` (string, required): "Path to AnimationPlayer node relative to scene root"
  - `animation_name` (string, required): "Name of the Animation to modify"
  - `length` (number, optional): "Animation duration in seconds"
  - `loop_mode` (string, optional): "Loop mode: none, linear, or pingpong"
  - `step` (number, optional): "Time step for snapping in the editor (e.g. 0.1)"
- required: ["player_path", "animation_name"]

All tools use `min_version: {4, 3, 0}`.
