   - Description: `"Batch set theme overrides on a Control node. Supports colors (hex or Color()), font sizes (int), and constants. Override type is auto-detected from key names or can be specified explicitly."`
   - Properties:
     - `node_path` (string, required): `"Path to Control node relative to scene root"`
     - `overrides` (object, required): `"Key-value pairs of theme overrides. Keys are theme override names (e.g., font_color, font_size). Values are strings: hex colors (#ff0000), integers (16), or Color() constructors."`
   - Required: `{"node_path", "overrides"}`

3. **create_stylebox** (UISYS-03):
   - Description: `"Create a StyleBoxFlat resource and apply it as a theme override on a Control node. Supports bg_color, corner_radius, border_width, border_color, content_margin, shadow, and anti-aliasing properties."`
   - Properties:
     - `node_path` (string, required): `"Path to Control node relative to scene root"`
     - `override_name` (string, required): `"Theme override name to apply the StyleBox to (e.g., panel, normal, hover, pressed, disabled, focus)"`

     - `corner_radius` (integer, optional): `"Corner radius applied to all 4 corners. Default: 0"`

     - `corner_radius_bottom_right` (integer, optional): `"Bottom-right corner radius (overrides corner_radius)"`
     - `corner_radius_bottom_left` (integer, optional): `"Bottom-left corner radius (overrides corner_radius)"`
     - `border_width` (integer, optional): `"Border width applied to all 4 sides. Default: 0"`
     - `border_color` (string, optional): `"Border color as hex or Color(). Default: #cccccc"`
     - `content_margin` (number, optional): `"Content margin applied to all 4 sides. Default: -1 (auto)"`
     - `shadow_color` (string, optional): `"Shadow color as hex or Color()"`
     - `shadow_size` (integer, optional): `"Shadow size in pixels. Default: 0"`
     - `anti_aliased` (boolean, optional): `"Enable anti-aliasing. Default: true"`
   - Required: `{"node_path", "override_name"}`

4. **get_ui_properties** (UISYS-04):
   - Description: `"Query a Control node's UI-specific properties: anchors, offsets, size flags, minimum size, pivot offset, grow direction, focus neighbors, and layout direction. For focus neighbor setting, use set_node_property with focus_neighbor_left/top/right/bottom."`
   - Properties:
     - `node_path` (string, required): `"Path to Control node relative to scene root"`
   - Required: `{"node_path"}`

5. **set_container_layout** (UISYS-05):
   - Description: `"Configure a Container node's layout parameters. For BoxContainer: alignment and separation. For GridContainer: columns and h/v separation. Optionally batch-set all direct children's size_flags."`
   - Properties:
     - `node_path` (string, required): `"Path to Container node relative to scene root"`
     - `separation` (integer, optional): `"Space between children in pixels (BoxContainer/GridContainer)"`
     - `h_separation` (integer, optional): `"Horizontal separation for GridContainer"`
     - `v_separation` (integer, optional): `"Vertical separation for GridContainer"`
     - `alignment` (string, optional): `"BoxContainer alignment: begin, center, or end"`
     - `columns` (integer, optional): `"Number of columns for GridContainer"`
     - `child_size_flags_horizontal` (string, optional): `"Batch-set all direct children's horizontal size flags: shrink_begin, fill, expand, expand_fill, shrink_center, shrink_end"`
     - `child_size_flags_vertical` (string, optional): `"Batch-set all direct children's vertical size flags: shrink_begin, fill, expand, expand_fill, shrink_center, shrink_end"`
   - Required: `{"node_path"}`

6. **get_theme_overrides** (UISYS-06 support):
